# Malaysia Radar Bot

Telegram bot for Malaysia monitoring: news, weather, currency, calendar and daily reports.

## Commands

- `/start` ‚Äî show bot status and chat_id
- `/weather` ‚Äî weather for target cities
- `/currency` ‚Äî MYR exchange rates
- `/news` ‚Äî Malaysia and city news
- `/calendar` ‚Äî upcoming holidays and local events
- `/report` ‚Äî full daily report
- `/help` ‚Äî command list

## Setup

1. Copy `.env.example` to `.env`.
2. Add your Telegram bot token to `.env`.
3. Install requirements.
4. Run `python3 bot.py`.

Never commit `.env` to GitHub.
